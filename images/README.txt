BILDER HIER ABLEGEN
====================

Lege deine Fotos in diesen Ordner. Sobald eine Datei hier liegt und
GENAU so heißt wie unten, erscheint sie automatisch auf der Seite —
du musst nichts im Code ändern.

Erwartete Dateinamen:

  tomekk-portrait.jpg   ->  Vita-Bereich (Porträt, am besten Hochformat ~4:5)
  gallery-1.jpg         ->  Galerie "Studio & Stage"
  gallery-2.jpg         ->  Galerie
  gallery-3.jpg         ->  Galerie
  gallery-4.jpg         ->  Galerie

Tipps:
  - Format JPG oder WebP
  - Lange Kante ~1600 px reicht völlig
  - Pro Bild moeglichst < ~300 KB (sonst laedt die Seite langsam)
  - Solange ein Bild fehlt, zeigt die Seite an der Stelle einen
    markierten Platzhalter mit dem erwarteten Dateinamen.

Hinweis: Der Hero ganz oben nutzt bewusst die Vinyl-Grafik und
braucht KEIN Foto. Wenn du dort lieber ein Bild willst, sag Bescheid.

Rechte nicht vergessen: nur Fotos verwenden, fuer die ihr die
Nutzungsrechte habt (Fotograf!).
